# COUNTBLANK

> Counts the number of blank cells within a range

## Syntax
COUNTBLANK(range)

TheCOUNTBLANKfunction syntax has the following arguments:

RangeRequired. The range from which you want to count the blank cells.

## Examples
- Copy the example data in the following table, and paste it in cell A1 of a new Excel worksheet.

- Table:
Data | Data | 
 |  | 
6 |  | 
 | 27 | 
4 | 34 | 
Formula | Description | Result
=COUNTBLANK(A2:B4) | Counts empty cells in the range above. | 2
 |  | 

- Note:Copying from some browsers may add an apostrophe to the beginning of the formula when it's pasted in Excel. If the formula does not show a result, select that cell, pressF2, delete the apostrophe, and then pressEnter.
