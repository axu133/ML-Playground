# SMALL

> Returns the k-th smallest value in a data set

## Syntax
SMALL(array, k)

The SMALL function syntax has the following arguments:

ArrayRequired. An array or range of numerical data for which you want to determine the k-th smallest value.KRequired. The position (from the smallest) in the array or range of data to return.
