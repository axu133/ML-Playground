# LOGNORM.INV

> Returns the inverse of the lognormal cumulative distribution

## Syntax
LOGNORM.INV(probability, mean, standard_dev)

The LOGNORM.INV function syntax has the following arguments:

ProbabilityRequired. A probability associated with the lognormal distribution.MeanRequired. The mean of ln(x).Standard_devRequired. The standard deviation of ln(x).
