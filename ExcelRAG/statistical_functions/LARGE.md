# LARGE

> Returns the k-th largest value in a data set

## Syntax
LARGE(array, k)

The LARGE function syntax has the following arguments:

ArrayRequired. The array or range of data for which you want to determine the k-th largest value.KRequired. The position (from the largest) in the array or cell range of data to return.

## Examples
- Copy the example data in the following table, and paste it in cell A1 of a new Excel worksheet. For formulas to show results, select them, press F2, and then press Enter. If you need to, you can adjust the column widths to see all the data.

- Table:
Data | Data | 
3 | 4 | 
5 | 2 | 
3 | 4 | 
5 | 6 | 
4 | 7 | 
Formula | Description | Result
=LARGE(A2:B6,3) | 3rd largest number in the numbers above | 5
=LARGE(A2:B6,7) | 7th largest number in the numbers above | 4
