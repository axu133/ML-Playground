# T.DIST.RT

> Returns the Student's t-distribution

## Syntax
T.DIST.RT(x,deg_freedom)

The T.DIST.RT function syntax has the following arguments:

XRequired. The numeric value at which to evaluate the distribution.Deg_freedomRequired. An integer indicating the number of degrees of freedom.
