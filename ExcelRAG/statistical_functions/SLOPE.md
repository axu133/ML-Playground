# SLOPE

> Returns the slope of the linear regression line

## Syntax
SLOPE(known_y's, known_x's)

The SLOPE function syntax has the following arguments:

Known_y'sRequired. An array or cell range of numeric dependent data points.Known_x'sRequired. The set of independent data points.
