# FREQUENCY

> Returns a frequency distribution as a vertical array

## Syntax
FREQUENCY(data_array, bins_array)The FREQUENCY function syntax has the following arguments:data_arrayRequired. An array of or reference to a set of values for which you want to count frequencies. If data_array contains no values, FREQUENCY returns an array of zeros.bins_arrayRequired. An array of or reference to intervals into which you want to group the values in data_array. If bins_array contains no values, FREQUENCY returns the number of elements in data_array.
