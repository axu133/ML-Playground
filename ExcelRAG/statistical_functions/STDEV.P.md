# STDEV.P

> Calculates standard deviation based on the entire population

## Syntax
STDEV.P(number1,[number2],...)

The STDEV.P function syntax has the following arguments:

Number1Required. The first number argument corresponding to a population.Number2, ...Optional. Number arguments 2 to 254 corresponding to a population. You can also use a single array or a reference to an array instead of arguments separated by commas.
