# PERCENTILE.EXC

> Returns the k-th percentile of values in a range, where k is in the range 0 to 1, exclusive.

## Syntax
=PERCENTILE.EXC(array,k)The PERCENTILE.EXC function syntax has the following arguments:ArgumentDescriptionarrayRequiredThe array or range of data that defines relative standing.kRequiredThe array or range of data that defines relative standing.
